## 💡 こだわったポイント

- **スムーススクロール** — `querySelectorAll` と `scrollIntoView` を使い、ナビリンクのクリックで各セクションへなめらかに移動できるよう JavaScript で実装しました
- **Worksカードのホバーアニメーション** — CSS の `transition` と `translateY` を使い、カードに触れると浮き上がる動きをつけ、クリックしたくなるUIを意識しました
- **スキルバー** — 各スキルの習熟度を視覚的に伝えるため、CSSでバーグラフを実装しました
- **レスポンシブデザイン** — `@media (max-width: 600px)` を使い、スマホでも崩れないレイアウトにしました
- **カラーブランディング** — オレンジ（`#FF9F1C`）をアクセントカラーとして統一し、ロゴ・ボタン・タグ・セクションラインに一貫して使用しています
